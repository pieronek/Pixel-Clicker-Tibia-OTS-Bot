\## üöÄ Features



\* \*\*Vitals Monitoring:\*\* Auto-heal or use potions based on pixel color changes (HP/Mana).

\* \*\*Mana Burn Logic:\*\* Special trigger for actions when mana/energy is above a certain threshold (Blue > 100).

\* \*\*Timed Rotation:\*\* Independent timers for spells and food (Blind cast).

\* \*\*Profile System:\*\* Save and load different configurations as `.json` files.

\* \*\*Multi-Tab UI:\*\* Organized interface with dedicated "How To" and "Spell CDs" sections.



\## üõ† Installation \& Usage



1\.  \*\*Run the App:\*\* Open `main.exe` (run as Administrator for best compatibility).

2\.  \*\*Pick Pixels:\*\* Click the \*\*Pick\*\* button, hover over your HP/Mana bar, and press `ENTER`.

3\.  \*\*Assign Binds:\*\* Choose the key (1-9, F1-F12) to be pressed.

4\.  \*\*Set Timers:\*\* Adjust the loop interval (ms) and cooldowns (s) for timed spells.

5\.  \*\*Start:\*\* Press \*\*START\*\* to begin the loop. Press `ESC` or \*\*STOP\*\* to abort.



> \*\*Note:\*\* Ensure your Windows display scaling is set to 100% for accurate pixel picking.
